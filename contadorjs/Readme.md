# Contador con JavaScript

Este proyecto es un contador básico hecho con **HTML, CSS y JavaScript**.  
Incluye funciones para aumentar, disminuir, reiniciar y guardar valores en LocalStorage.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript Vanilla

## Funciones
- Incrementar contador
- Decrementar contador
- Reiniciar contador
- Guardar valor en historial con LocalStorage
